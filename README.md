# Vida en Cristo — Life in Christ

A beautiful, animated religious landing page built with **Next.js 16**, **React 19**, and **Tailwind CSS v4**.

## Features

- 🌐 **Multi-language** — English, Spanish (default), and French
- 🙏 **Prayers** — Lord's Prayer, Hail Mary, Glory Be, Apostles' Creed
- 🌹 **Rosary** — All four sets of mysteries with interactive guide
- 🕊️ **Works of Mercy** — Corporal and Spiritual Works
- 🖼️ **Gallery** — Sacred images
- ✨ **Animations** — Particles, fade-ins, light rays, scroll effects (Framer Motion)
- 🎨 **Divine theme** — Deep blue/purple/gold celestial color palette

## Tech Stack

- **Next.js 16.1** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **Framer Motion** (animations)
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Build

```bash
npm run build
npm start
```
